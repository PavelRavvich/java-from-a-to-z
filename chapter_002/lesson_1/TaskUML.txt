Task - класс шаблон заявки, включая ее возможные правки.
Add - создает динамичный массив и добавляет в него объекты типа Task.
Edit - находит по имени задачу за счет перебора массива и меняет ее содержание.
Commit - заполняет поле комментария использует переопределенный метод из Edit
Remove - находит по имени задачу за счет перебора массива и удаляет всю ячейку использует переопределенный метод из Edit.
View - печатает массив с заявками по прядку как он есть.
ViewFilter - печатает массив с заявками предварительно как-то сортеруя наверно тоже должен переопределять метод view из класса View.

Может это все не правильно, но пока ничего другого в голу не приходт...
